## 🔄 Git Workflow

Every assignment follows this version control process:

```bash

cd C:\CSD\CSD325

# 2. Check current status
git status

# 3. Stage all changes
git add --all

# 4. Commit with a descriptive message
git commit -m "Module X - Assignment description MM/DD/YY"

# 5. Push to GitHub
git push origin main
```
